# AI Complaint Categorization System

React dashboard for AI-powered complaint categorization and admin management.

## Features
- Admin dashboard with analytics charts
- Complaints view & categorization
- Responsive Shadcn UI + Tailwind
- TanStack Query for data fetching
- TypeScript + Vite

## Setup
```bash
bun install
bun dev
```

## Structure
```
src/pages/
  - AdminDashboard.tsx
  - AdminComplaints.tsx
```

Visit [localhost:5173](http://localhost:5173)
