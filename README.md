# AI Complaint Categorization System

## Overview
React + Vite + TypeScript + Shadcn UI dashboard for AI-powered complaint categorization. Features admin dashboard and complaints management.

## Tech Stack
- React 18
- Vite
- TypeScript
- Shadcn UI components
- TanStack Query
- Tailwind CSS
- Recharts

## Quick Start
```bash
bun install
bun dev
```

## Structure
- `src/pages/`: Dashboard and Complaints pages
- `public/`: Static assets

Built with ❤️ using modern tools.
