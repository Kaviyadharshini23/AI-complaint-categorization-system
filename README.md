# AI Complaint Categorization System 🚀

## Overview
Advanced web dashboard for **AI-powered complaint categorization**. Built with modern React ecosystem, this system allows admins to manage, view, and analyze complaints categorized using AI models.

**Key Features:**
- Admin Dashboard with charts (Recharts)
- Complaints listing & management
- Responsive Shadcn UI components
- Real-time data with TanStack Query
- Dark/Light theme support
- TypeScript end-to-end

## 🎯 AI Integration
- Processes complaints through AI classification
- Categories: [Technical, Billing, Service, Fraud, Other]
- Dashboard analytics on complaint trends

## Tech Stack
```
Frontend: React 18 + Vite + TypeScript
UI: Shadcn UI + Tailwind CSS + Lucide Icons
Charts: Recharts
State: TanStack Query + React Hook Form + Zod
Routing: React Router
Testing: Vitest + Playwright + Testing Library
```

## 🚀 Quick Start

1. **Clone & Install**
```bash
git clone https://github.com/Kaviyadharshini23/AI-complaint-categorization-system.git
cd AI-complaint-categorization-system
bun install  # or npm/yarn/pnpm install
```

2. **Development**
```bash
bun dev
```
Open [http://localhost:5173](http://localhost:5173)

3. **Build & Preview**
```bash
bun build
bun preview
```
